Projet 1 - Réservation de séjour et gestion des colloques (CodeIgniter)

Organisation : CVVEN (Comité pour les Villages de Vacances de l'Éducation Nationale)

Période : année 2025/2026

Contexte : conception d'une application web de gestion et réservation de locations, avec architecture MVC (Modèle-Vue-Contrôleur) et base PostgreSQL.

Fonctionnalités réalisées

  • Authentification des utilisateurs et gestion des accès
  • Consultation des locations disponibles et réservation
  • Gestion des clients et des colloques (création, modification, suppression)
  • Paramètres utilisateur : thème, notifications, sécurité et confidentialité

Environnement technique

  • Langages : PHP, SQL, HTML, CSS, JavaScript
  • Framework : CodeIgniter
  • Base de données : PostgreSQL (administration via PhpPgAdmin)
  • IDE : NetBeans
  • Tests sécurité : Burp Suite

Compétences E5 mobilisées

  • Concevoir et développer une solution applicative
  • Assurer la maintenance corrective ou évolutive
  • Gérer les données

Livrables : code source, base de données, application opérationnelle et documentation de conception/réalisation/tests.

Projet 2 - Application de ticketing d'assistance (Python Flask)

Organisation : CVVEN (Comité pour les Villages de Vacances de l'Éducation Nationale)

Période : année 2025/2026

Contexte : développement d'une application web de suivi des incidents pour structurer les demandes d'assistance et d'évolution.

Fonctionnalités réalisées

  • Connexion sécurisée avec gestion des rôles (utilisateur / administrateur)
  • Tableau de bord de suivi des tickets (ouverts, en cours, fermés)
  • Création de ticket avec catégorie, localisation et niveau de priorité
  • Mise à jour du statut, assignation à un technicien et notes de résolution
  • Mots de passe hachés en base pour renforcer la sécurité

Environnement technique

  • Langage : Python
  • Framework : Flask
  • Base de données : SQLite
  • IDE : Visual Studio Code
  • Outils projet : Trello, GitHub

Compétences E5 mobilisées

  • Concevoir et développer une solution applicative
  • Assurer la maintenance corrective ou évolutive
  • Gérer les données

Livrables : code source, base SQLite, application fonctionnelle en local et documentations utilisateur/gestionnaire/développeur.
